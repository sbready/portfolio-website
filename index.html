About Me

I have always had an active imagination. When I was young I wanted to be an inventor. I had no real notion of what I wanted to invent, but I knew that was the direction that my life would take me. Although I could not draw, I would feebly try to put down on paper some form of what my mind was creating... clocks, robots, drones, watches, computers, it didn't really matter as long as I thought it was an improvement. I have grown older, but the drive to create has only grown stronger. Although I still cannot draw, my mind still creates. For "thought is the original source of all wealth, all success, all material gain, all great discoveries and inventions, and of all achievement." This drive to create has led me to the world of computer science. As a developer, I embrace constant challenges and innovation. You can follow some of my work here!

Personally Developed Web App

The Candidate Review

The Candidate Review was created as an educational tool catered for the voting masses. This allows upcoming candidates to create profiles of their positions based on the position they are seeking.

Technology Used:

React | Redux | NodeJS | MassiveJS | Material UI | PostgreSQL | AmazonS3 | Auth0

  • React implemented to help display changing information across reusable components.
  • Material UI to create stepper processes for user and candidate profile create.
  • Heavily integrated Redux throughout site to store candidate and user data.
  • Data heavy creating atmosphere to better understand data parsing and data types.
  • RESTful API integration throughout backend and on Redux reducers.

Group Developed Web App

Oatmeal Gamification

Oatmeal is an interactive CRM/Gamification platform built to motivate and stimulate user attention and focus through live-updating interactions.

Technology Used:

React | Redux | NodeJS | MassiveJS | Semantic UI | PostgreSQL | AmazonS3 | Socket.io | lodash

  • React implemented to create interactive and independent components to display information based on challenge or badge variables designed by the user.
  • Responsive with use of Media Queries and Semantic UI elements creating a seamless user interface on desktop, tablet, or mobile devices.
  • Mini-modeled CRM created in order to have interactive data response on the gamification platform.
  • Socket.io integrated to display live-updating data split by overall aggregated information and rooms detailing individual standings
  • Images interact with Amazon S3 services through automatic file upload/download components
